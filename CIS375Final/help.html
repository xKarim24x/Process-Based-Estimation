Instructions for Process Based Estimation tool

The following instructions will list the different ways that your able to use and interact with the PBE tiny tool

To get started with the Process based Calculator Enter a function name and select the "Add Column" button to add your first activity.

Once this is done, enter the the activity name and corresponding hours.

To add new functions select the "Add Row" button

To add new Activities select the "Add Column" button

To remove functions from the calculator select the "delete Row" button
To remove Activities from the calculator select the "Delete Column" button

To calculate your function point total select the "Calculate" button
